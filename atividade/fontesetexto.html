Textos

Os textos na WEB são colocados na pagina atravez de algumas tags no HTML como: "p" para paragrafos "h1,h2,h3,h4,h5,h6" para titulos e outros conteudos também pode se alterar ou destacar uma parte do texto como por exemplo por em italico com a tag "i" assim como a palavra "italico", a tag "em" também para destacar uma parte do texto como nesta frase.


Tag span é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ), ou para compartilhar valores de atributos como lang.


Fontes

As fontes de uma pagina WEB podem ser alteradas, atravez de propriedades no CSS como "font-family". Podem também serem exportadas de fora por fontes encontradas na internet.
