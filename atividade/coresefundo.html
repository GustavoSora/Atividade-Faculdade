FUNDO

A cor de fundo de uma página web pode ser alterado utilizando a propriedade "background-color", através de uma tag style ou de um arquivo CSS externo. Utilizando esta página como exemplo é possivel ver a utilização do "background-color" na cor cinza (58,58,58). Também e possivel utilizar imagens para alterar o fundo da página, utilizando o atributo "background-image".




Cores

O atributo "color" é utilizado também para mudar a cor da fonte, como por exemplo a palavra "cores" e o titulo da página, que estão verdes pois foi aplicado a propriedade "color:green" no arquivo CSS.
